# Learn Java & DSA

This repository contains my solutions and practice code for various Java programming concepts and Data Structures & Algorithms (DSA) problems.

## 🗂️ Project Structure

- **`Java Basics/`**: Core Java concepts, syntax, OOPS basics, and simple programs.
- **`Data Structures/`**: Implementations and problems related to:
  - Arrays
  - Linked Lists
  - Stacks & Queues
  - Trees & Tries
  - Graphs
  - Hash Tables
- **`Algorithms/`**:
  - Searching & Sorting
  - Recursion & Backtracking
  - Dynamic Programming (DP)
  - Greedy Algorithms
- **`Competitive Programming/`**: Solutions to problems from platforms like LeetCode, HackerRank, GeeksforGeeks, etc.

## 🚀 How to Run

To compile and run any Java file:

```bash
javac FileName.java
java FileName
```
